# Panduan Instalasi Python

Python adalah bahasa pemrograman yang serbaguna dan populer yang digunakan dalam berbagai proyek pengembangan perangkat lunak. Berikut adalah panduan langkah demi langkah untuk menginstal Python di sistem Anda.

## Langkah 1: Unduh Python

Unduh installer Python dari situs resmi Python di [python.org](https://www.python.org/downloads/). Pilih versi Python yang sesuai dengan sistem operasi Anda (Windows, macOS, atau Linux).

## Langkah 2: Mulai Instalasi

### Windows
1. Buka file installer Python yang telah Anda unduh.
2. Pastikan Anda memeriksa kotak "Add Python to PATH" untuk memudahkan penggunaan Python dari Command Prompt.
3. Klik "Install Now" untuk memulai instalasi.

### macOS
1. Buka file installer Python yang telah Anda unduh.
2. Ikuti panduan instalasi dengan mengklik "Continue" dan "Install" saat diperlukan.
3. Tunggu hingga instalasi selesai.

### Linux
1. Buka terminal dan navigasikan ke direktori di mana Anda menyimpan file installer Python.
2. Jalankan perintah berikut untuk menginstal Python: `sudo ./python-installer-filename.run`. Gantilah `python-installer-filename.run` dengan nama file installer Python yang sesuai.

## Langkah 3: Verifikasi Instalasi

Setelah instalasi selesai, buka terminal atau Command Prompt dan jalankan perintah berikut untuk memeriksa versi Python yang terinstal:
